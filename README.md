Data for Amharic speech recognition. Use together with [melanet](https://github.com/tilayealemu/melanet).

To extract:
```
git clone git@github.com:tilayealemu/melanet-data
cd melanet-data/data
cat data.tar.gz.* > data.tar.gz
tar xzf data.tar.gz
```
